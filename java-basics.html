Java Basics Shortnotes

       

1. Java Introduction

       
               
  • Java is a high-level, object-oriented programming language.
    •            
  • Platform independent (WORA: Write Once, Run Anywhere).
    •            
  • Syntax similar to C/C++.
    •        
       

2. Data Types

       
               
  • Primitive: int, float, double, char, byte, short, long, boolean.
    •            
  • Non-Primitive: String, Array, Class, Object, Interface.
    •        
       

3. Variables

       
               
  • Declaration: int a;
    •            
  • Initialization: a = 10;
    •            
  • Types: local, instance, static.
    •        
       

4. Basic Input/Output

       
               
  • Input: Scanner sc = new Scanner(System.in);
    •            
  • Output: System.out.println("Hello");
    •        
       

5. Control Statements

       
               
  • if, else if, else: Conditional branching.
    •            
  • switch: Multiple selections using cases.
    •            
  • for, while, do-while: Looping constructs.
    •            
  • break, continue: Loop control.
    •        
       

6. Functions/Methods

       
               
  • Method declaration: returnType methodName(parameters) { }
    •            
  • Static methods: static void main(String[] args)
    •        
       

7. Arrays

       
               
  • Declaration: int[] arr = new int[5];
    •            
  • Initialization: arr[0] = 1;
    •            
  • Length: arr.length
    •        
       

8. Classes & Objects

       
               
  • Class declaration: class Student { }
    •            
  • Object creation: Student s = new Student();
    •        
       

9. Constructors

       
               
  • Default and parameterized constructors.
    •            
  • Constructor name = class name, no return type.
    •        
       

10. Inheritance

       
               
  • class B extends A
    •            
  • Single, multilevel, and hierarchical inheritance.
    •            
  • super keyword for parent class reference.
    •        
       

11. Polymorphism

       
               
  • Method overloading: Same method name, different parameters.
    •            
  • Method overriding: Subclass provides specific method implementation.
    •        
       

12. Encapsulation

       
               
  • Wrapping data (variables) and code (methods) together.
    •            
  • Private variables, public getters/setters.
    •        
       

13. Abstraction

       
               
  • Hiding implementation, showing functionality.
    •            
  • Abstract class: abstract class Shape {}
    •            
  • Interface: interface Animal {}
    •        
       

14. Exception Handling

       
               
  • try-catch-finally
    •            
  • throw, throws keywords
    •            
  • Common exceptions: NullPointerException, ArrayIndexOutOfBoundsException
    •        
       

15. Useful Java Shortcodes

       
               
  • Print: System.out.println()
    •            
  • Array Sorting: Arrays.sort(arr);
    •            
  • Array to ArrayList: new ArrayList<>(Arrays.asList(arr));
    •            
  • String to int: Integer.parseInt(str);
    •            
  • Int to String: String.valueOf(num);
    •        
       

16. Common Interview Points

       
               
  • Difference between == and .equals()
    •            
  • Immutable String objects
    •            
  • Access modifiers: public, private, protected, default
    •            
  • Static keyword importance
